# Project Nebula

**Stellar Constellation Code Mapper**

Project Nebula is a stunning, deep-space themed visualization tool that maps Github repository commit histories. Instead of traditional lists or grids, Nebula transforms codebases into an interactive 2D star constellation. Each commit forms a node connected across an interactive star map, complete with highly-responsive glassmorphism details.

## Features
- Interactive SVG mathematical mapping of the repository commits.
- Zero-dependency standalone prototype natively available through the browser (`standalone.html`).
- Polished Tailwind CSS designs featuring deep space imagery, glowing node halos, and micro-animations.
- Full React / Vite Architecture for scalable development.

## Getting Started

Make sure you have Node.js installed. In your terminal, verify you are in the project root and run:

```bash
npm install
npm run dev
```

Visit `http://localhost:5173` to explore your codebase through the stars!
